Tag validateAllOrNone

  • Description:
    o:validateAllOrNone validates validates if at least ALL of the given UIInput components have been filled out or that NONE of the given UIInput components have been filled out. The default message is
    {0}: Please fill out all or none of those fields

    For general usage instructions, refer validateAll tag documentation.
