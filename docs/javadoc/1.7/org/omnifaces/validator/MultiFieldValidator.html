      • validateValues

        boolean validateValues(FacesContext context,
                     List<UIInput> components,
                     List<Object> values)
        Perform the validation on the collected values of the input components and returns whether the validation is successful. Note: this validator does NOT throw ValidatorException! This job is up to the ValidateMultipleFields implementation who will call this method.
        Parameters:
        context - The faces context to work with.
        components - The input components whose values are to be validated.
        values - The values of the input components to be validated.
        Returns:
        true if validation is successful, otherwise false (and thus show the message).
