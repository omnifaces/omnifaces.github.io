org.omnifaces.util

Class Platform

  • java.lang.Object
    • org.omnifaces.util.Platform

  • public final class Platform
extends Object
    This class provides access to (Java EE 6) platform services from the view point of JSF.

    Note that this utility class can only be used in a JSF environment and is thus not a Java EE general way to obtain platform services.
